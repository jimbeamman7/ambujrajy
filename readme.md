### KcptunGUI是什么?
KcptunGUI是kcptun的Windows用户界面,用于帮助用户简易方便的配置参数以及管理服务.

已包含kcptun v20160820,无需额外下载.

发布版本可能不是最新版,最新版请自行下载编译.

![](http://i1.buimg.com/1949/c569f5074eca2af4.png)

### 需求
- [.Net运行时最新版](http://go.microsoft.com/fwlink/?LinkId=780601)
- 操作系统最低Windows7
